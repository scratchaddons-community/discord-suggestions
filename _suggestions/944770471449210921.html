Ok so hear me out- What if, when making a custom block, we had the option to make it appear in a certain palette (Thus, making the block the same color of that palette) And after making the code for a block, we could have an option to hide the code that defines that block as well as show the code when right clicking on the block in the palette. Speaking of right clicking the block in the palette, there should be an option for "locking" the block. This would make the block appear in every project you go to without having to import the code. But we should also have the ability to show and hide blocks of any kind, whether we coded them or not. So when we right click on the palette, we have a list of all the blocks and we can click on any one of them to show or hide it. This whole idea is a lot to take in and can be all over the place, so I made some images to recreate what I'm going for.
